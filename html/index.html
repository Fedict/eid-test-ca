eID Test CA

This docker container provides a PKI infrastructure which was written to emulate the official Belgian eID environment. This webserver provides the management interface and the CRLs; the OCSP server can be found running on its own port (8888 by default).

While every effort has been taken to ensure that this test environment behaves in the same way as the official PKI infrastructure, mistakes are possible. If you encounter a difference, please file a bugreport, and we will look into it. The source code of this container is also available through the same github repository.
